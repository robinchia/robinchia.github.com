之前先列另一个文章出来：

  1. 避免Flash挡住Div层的方式：增加 ，另外增加
  2. select和flash挡住div，可以采取移除节点的方式防止挡住
  3. select和flash采用display:none的方式防止挡住
  4. 在select和flash上面加一层隐藏的iframe对象，因为iframe对象跟select和flash界面显示平级，然后把目标div放在iframe上面
